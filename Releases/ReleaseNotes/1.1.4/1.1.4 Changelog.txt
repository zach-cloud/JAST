- The "Merge Code" window now supports generic templates.
- These templates are located in the "JAST/templates" folder
- Function and Trigger templates have been created
- You can add your own template if you follow the example.

[WC3Edit Release Only]

- Adding NZCP now allows for a custom arrow & name activator
- If available, the "Merge Code" window prioritizes NZCP.
